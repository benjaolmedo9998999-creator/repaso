# repaso

### 1. Variables y Aritmética Básica
Este script demuestra cómo Python gestiona la memoria mediante variables y realiza operaciones matemáticas simples.
> **Concepto clave:** Almacenamiento de datos y uso de la función `print()` con múltiples argumentos.


### 2. Lógica de Decisión (`if-else`)
Aquí implemento un control de flujo para validar condiciones. Es el pilar de la interactividad en un programa.
* **Operador de comparación:** `>=` (mayor o igual).
* **Bloques de código:** Uso de la indentación para separar la lógica de éxito y de error.


### 3. Estructuras de Repetición (Bucles)
Comparativa entre los dos tipos de ciclos principales en Python:
| Ciclo | Uso en el ejemplo | Lógica |
| :--- | :--- | :--- |
| **FOR** | Iteración sobre un rango | Recorre del 1 al 5 automáticamente. |
| **WHILE** | Control mediante condición | Se ejecuta mientras el contador sea mayor a 0. |


### 4. Modularización con Funciones
Uso de la palabra reservada `def` para crear código reutilizable. Esto evita la duplicación y mejora la legibilidad.
* **Parámetros:** La función recibe un `nombre` y lo procesa dentro de un mensaje personalizado.
* **Llamadas:** Se demuestra cómo invocar la función con diferentes argumentos (`Alan`, `Marta`).
